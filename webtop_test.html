Webtop

This is a simple web based desktop that uses just one client side javaScript, and CSS file, with an HTML file like this one. You can put regular HTML content under the "app" windows, like you see here.

We looked at other Web Desktops at https://en.wikipedia.org/wiki/Web_desktop.

You can get the source to this web desktop project at on github at https://github.com/vtwireless/webtop.
